The Lead Tracker provides a structured approach to managing leads by breaking the process into stages and sub-stages. This makes it easy to see where each lead stands, what actions are required, and how to move them forward.

![Lead Tracker Nav Web](/images/lead_tracker_nav.webp)

**Key Definitions**

| **Term**               | **Meaning**                                                                |
| :--------------------- | :------------------------------------------------------------------------- |
| **Lead**               | A potential patient record with key metadata (e.g., referral type, stage). |
| **Reporting category** | Groups stages for reporting purposes (e.g., Converted, In Progress).       |
| **Stage**              | The main phase a lead passes through (e.g., New, Contacted, Converted).    |
| **Sub-stage**          | Specific steps within a stage for more granular tracking.                  |
| **Referral type**      | Categorizes the source of a lead (e.g., Provider, Online Form, Fax).       |
| **Treatment Declined** | Marks leads that declined treatment, useful for reporting and tracking.    |

## Managing Leads

### Creating a Lead

<Card title>
  1. Click **\+ New**.
  2. Enter patient's First Name, Last Name, and DOB.
  3. Select referral type.
  4. Assign a facility and referring provider (optional).
  5. Choose referral stage and sub-stage.
  6. Add first/last contact dates.
  7. Assign responsibility to a staff member.
  8. Link patient - leads can be linked one-time to a **new** or **existing patient** or not be linked to any patient (if there is not enough information). While linking, Air will automatically suggest patients.
  9. Click **Save**.

  ![Lead New Web](/images/lead_new.webp)
</Card>

### Updating a Lead Status

<Card title>
  - Click on the pencil icon on the right of a lead
  - Update referral stage, sub-stage and last contacted details.
  - **Scheduled status:** To move a lead status to scheduled you can update the status and click on the patient name to open the patient's profile and schedule an appointment.
  - **Treatment declined:** Click Treatment Declined on the bottom right in red of the edit lead pop-up and select a reason. Leads marked declined can be tracked in the **Declined Treatment tab**. This status can be undone if needed.

  ![21 Denytreatmentleads Gi](/images/21_denytreatmentleads.gif)
</Card>

### Intelligent Fax Parsing

<Card title>
  If enabled, inbound faxes can automatically generate new leads.

  1. **Inbound Fax Processing** – The system analyzes incoming faxes using GPT to detect referral details.
  2. **Validation** – A lead is only created if First Name, Last Name, and DOB are detected.
  3. **Lead Creation** – The referral type is set to **Fax**, and the lead is placed in the first stage with a note: _“Automatically imported through inbound fax.”_
</Card>

## Configuring the Lead Tracker

### Lead Stages

<Card title>
  1. Go to **Lead Tracker \> Settings**.
  2. Click **\+ Add Stage** to create a new stage.
  3. Edit, rename, reorder, or delete stages as needed.
  4. Configure stage details:
     - Columns to display
     - Sub-stages
     - Reporting category
  5. Add sub-stages by clicking **\+ Add Sub-stage**.
  6. Customize sub-stages with names, colors, and order.
  7. Assign a reporting category (default = No Category).
  8. Click **Save Changes** to confirm.

### Referral Types

<Card title>
  1. In **Lead Tracker \> Settings**, open the **Referral Types** tab.
  2. Click **\+ Add New** to create a new referral type.
  3. Edit names with the pencil icon.
  4. Delete referral types if no longer needed.
  5. Click **Save Changes** to confirm.
</Card>

### Reporting Categories

<Card title>
  1. In **Lead Tracker \> Settings**, open **Stages**.
  2. Under **Reporting Category**, choose:
     - **Converted**
     - **In Progress**
     - **No Category** (default)
  3. Save changes.
</Card>

### Custom Landing Page

<Card title>
  You can collect leads directly from your website using a branded landing page.

  1. Go to **Lead Tracker \> Settings \> Landing Page**
  2. Toggle **Landing Page Active** ON.
  3. Configure required fields:
     - Header (max 30 chars)
     - Welcome message (max 255 chars)
     - Referral type
     - Stage assignment
     - Logo (PNG/JPG, max 4.2MB)
  4. Save changes.

  Patients visiting the page will submit details (name, DOB, phone, email, preferred facility), and new leads will be automatically created in the tracker.
