title: "Patient Appointment Details"
---

Review and manage patient appointments, chart notes, billing documents, and imported medical records.

To open a Patient's Profile: i) Click on the Top Tab of the Chart Note, ii) Open Patients Tab on the left navigation bar and select the desired patient.

### View chart notes for an appointment

<Card title=" ">
  Click on the **Appointments** tab at the top of the Patient Profile

  Click on the **square icon** (to the right of the calendar icon) to open the Chart Note for the desired appointment.in

  ![Pt Appt Past Web](/images/pt_appt_past.webp)
</Card>

### Filter appointments

<Card title=" ">
  Filter appointments by: Date of Appointment - Provider - Case - Facility - Status - Custom Tags

  ![16 Filter Pt Appointments Gi](/images/16_filter_pt_appointments.gif)
</Card>

### Download patient schedule

<Card title=" ">
  Select appointments by checking the box to the left of each appointment → Choose to **Download** or **Fax** the patient’s schedule.

  ![Appt Download Schedule Web](/images/appt_download_schedule.webp)
</Card>

**Import into Current Appointment Chart Note**

<Card title=" ">
  <Danger>

  </Danger>
  <img

    title="Import Doc In Scribe Web"
    style={{ width:"51%" }}
  />

  <Accordion title="Alternate method: Through Chart Note" defaultOpen>

    <img
      src="/images/import_doc_chart_note.webp"
      alt="Import Doc Chart Note Web"
      title="Import Doc Chart Note Web"

**Import Records for Past Appointments**

<Card title=" ">
  - Click the **\+ icon** to the right of the appointment → **Set Up Chart Note**.
  - Select the **Case, Appointment Type,** and **Clinical Note Type**.
  - Upload the document.

  ![17 Import Records Gi](/images/17_import_records.gif)

  **Note:** Select **For Documentation Only** when uploading.

  <img
    src="/images/for_doc_only.webp"
    alt="For Doc Only Web"
    title="For Doc Only Web"
    style={{ width:"37%" }}
  />

  This option ensures the imported chart note is **not billable**. It is especially useful when you need to manually add the Initial Evaluation while continuing the Plan of Care for a Progress Note or Re-Evaluation.
</Card>
