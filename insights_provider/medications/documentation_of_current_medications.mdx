## Description

Percentage of visits for which the eligible clinician attests to documenting, updating, or reviewing a list of current medications using all immediate resources available on the date of the encounter.

## How to track

**To document current medications in a Visit Note:**

1. Open the **Visit Note** and go to the **Medications** category.
2. Add the **Medications** and **Medication Review Attestation** sections (if they aren’t already on your note).

![Visit Note showing the Medications category and required sections](/images/insights_provider/medications/documentation_of_current_medications/documentation_of_current_medications_1.webp)

3. Under **Medication Review Attestation**, select the appropriate **Medication Review Status**.
4. If current medications were reviewed, select **Reviewed Current Medications**. This will be a **Performance Met**.

![Medication Review Attestation set to Reviewed Current Medications](/images/insights_provider/medications/documentation_of_current_medications/documentation_of_current_medications_2.webp)

5. If current medications were not reviewed, select **Did Not Review Current Medications**. This will be a **Performance Not Met**.

![Medication Review Attestation set to Did Not Review Current Medications](/images/insights_provider/medications/documentation_of_current_medications/documentation_of_current_medications_3.webp)

This will give you two options for **Reason**:

- **Emergency/Critical Situation**: considered an **Exception**

![Reason set to Emergency/Critical Situation](/images/insights_provider/medications/documentation_of_current_medications/documentation_of_current_medications_4.webp)

- **Other Reason**: remains **Performance Not Met**

![Reason set to Other Reason](/images/insights_provider/medications/documentation_of_current_medications/documentation_of_current_medications_5.webp)

## Criteria

- Report for every eligible encounter during the measurement period (each encounter is its own “episode”).

### FAQ

<Accordion title="What’s the difference between Performance Met, Performance Not Met, and Exception?">
  - **Performance Met**: You selected **Reviewed Current Medications**.
  - **Performance Not Met**: You selected **Did Not Review Current Medications** and chose **Other Reason**.
  - **Exception**: You selected **Did Not Review Current Medications** and chose **Emergency/Critical Situation** as the reason.
</Accordion>

<Accordion title="I don’t see Medication Review Attestation on my Visit Note. What should I do?">
  Add the **Medications** and **Medication Review Attestation** sections to your note, then complete the **Medication Review Status** field.
</Accordion>

<Accordion title="When should I use Emergency/Critical Situation as the reason?">
  Use **Emergency/Critical Situation** when an urgent or critical clinical situation prevents you from reviewing current medications during the encounter.
</Accordion>
