This measure tracks how often you document, update, or review a patient’s **current medications** during an eligible visit by completing a required attestation in the **Visit Note**.

## Overview

### <Icon icon="clipboard-check" iconType="duotone" color="#F9345F" size={23} /> What this measure tracks

**Documentation of Current Medications** is the percentage of **visits** where an eligible clinician attests to **documenting, updating, or reviewing** a list of **current medications** using **all immediate resources available on the date of the encounter**.

### <Icon icon="circle-dot" iconType="duotone" color="#F9345F" size={23} /> When a visit is counted

This measure is evaluated per eligible encounter:

- Each eligible encounter is its own “episode.”
- You should complete the attestation for **every eligible encounter** during the measurement period.

## How to complete the attestation in the Visit Note

### <Icon icon="notes-medical" iconType="duotone" color="#F9345F" size={23} /> Add the required sections

**To document current medications:**

1. Open the patient’s **Visit Note**.
2. Go to the **Medications** category.
3. Add the **Medications** section and the **Medication Review Attestation** section (if they aren’t already on the note).

### <Icon icon="check" iconType="duotone" color="#F9345F" size={23} /> Select a Medication Review Status

In **Medication Review Attestation**, select the appropriate **Medication Review Status** for the encounter.

| **Medication Review Status** | **How it counts** | **What to do next** |
| :--------------------------- | :---------------- | :------------------ |
| **Reviewed Current Medications** | **Performance Met** | Save/sign your note as usual. |
| **Did Not Review Current Medications** | **Exception** or **Performance Not Met** (depends on reason) | Select a reason for why medications were not reviewed. |

If you choose **Did Not Review Current Medications**, select one of the following reasons:

- **Emergency/Critical Situation**: counts as an **Exception**
- **Other Reason**: counts as **Performance Not Met**

<Tip>
  ✨**Smart Tip:** If your organization uses templates, add **Medication Review Attestation** to your default Visit Note template so you don’t have to add it each time.
</Tip>

## Reporting criteria

### <Icon icon="list-check" iconType="duotone" color="#F9345F" size={23} /> Reporting expectations

- Report for **every eligible encounter** during the measurement period.
- Each encounter is evaluated independently (one encounter = one episode).

### FAQ

<Accordion title="What counts as ‘reviewing current medications’ for this measure?">
  You should select **Reviewed Current Medications** when you have documented, updated, or reviewed the patient’s current medication list using all immediate resources available at the time of the encounter.
</Accordion>

<Accordion title="When should I use the ‘Emergency/Critical Situation’ reason?">
  Use **Emergency/Critical Situation** only when an urgent situation prevents medication review during that encounter. This reason is counted as an **Exception** for the measure.
</Accordion>

<Accordion title="If I didn’t review medications, will selecting ‘Other Reason’ still help my score?">
  No. **Other Reason** is counted as **Performance Not Met** for the measure. If medication review was not completed, choose the most accurate reason for the encounter.
</Accordion>

<Accordion title="Do I need to complete this for every visit?">
  You should complete the attestation for every **eligible encounter** during the measurement period. If an encounter is not eligible, it will not be counted for this measure.
</Accordion>
