This measure tracks the percentage of eligible visits where you **attest** that you documented, updated, or reviewed the patient’s **current medications** using all immediate resources available on the date of the encounter.

### <Icon icon="notes-medical" iconType="duotone" color="#F9345F" size={23} />  Where you document current medications in the Visit Note

You’ll complete this attestation inside the **Visit Note**, under the **Medications** category.

**To open the right area of the note:**

1. Open the patient’s **Visit Note**.
2. Navigate to the **Medications** category in the note.

### <Icon icon="circle-plus" iconType="duotone" color="#F9345F" size={23} />  Add the required sections (if they aren’t already on your note)

If you don’t see the sections below, add them to the note before you attest.

**To add the sections:**

1. In the **Medications** category, add the **Medications** section.
2. Add the **Medication Review Attestation** section.

**Note:** Practices can configure which sections appear by default. If you can’t add these sections, contact your practice admin.

### <Icon icon="clipboard-check" iconType="duotone" color="#F9345F" size={23} />  Complete the Medication Review Attestation

In **Medication Review Attestation**, select the option that matches what happened during the encounter.

- **Reviewed Current Medications** → **Performance Met**
- **Did Not Review Current Medications** → **Performance Not Met**, unless you document an exception reason
  - **Emergency/Critical Situation** → **Exception**
  - **Other Reason** → **Performance Not Met**

### FAQ

<Accordion title="What should I select if I reviewed the list and made updates?">
  Select **Reviewed Current Medications**. This indicates you reviewed and/or updated the patient’s current medications during the encounter and is counted as **Performance Met**.
</Accordion>

<Accordion title="What if the patient has no current medications?">
  If you verified the patient is not taking any medications, select **Reviewed Current Medications** (you still reviewed the medication list, even if the result is “none”).
</Accordion>

<Accordion title="When should I use the Emergency/Critical Situation reason?">
  Use **Emergency/Critical Situation** only when the clinical situation prevents you from reviewing current medications during the encounter. This is counted as an **Exception**.
</Accordion>

<Accordion title="Why am I seeing Performance Not Met for this visit?">
  The visit is counted as **Performance Not Met** when you select **Did Not Review Current Medications** and either:

  - You select **Other Reason**, or
  - No exception reason is documented
</Accordion>
