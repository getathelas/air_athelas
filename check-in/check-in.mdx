# **Manually check-in your patient**

<Card title=" ">
  Calendar → Click on Appointment → Check-in the patient

  You can change the appointment status to completed, no show, canceled or archived

  \<gif\> 

  **Alternate option**: Navigate to Patients tab on the left navigation bar → select Appointments Tab → select “Check in” (black square icon)

  <Danger>
    **Note:** In order to check in an Appointment a **<u>Case Type </u>**must be present. If there is no given case, the user will receive an error when trying to check in.
  </Danger>
</Card>
