### Change the Case Type for an appointment

**Use Existing Case Type**

<Card title=" ">
  \*\*Enter existing case type: \*\*Click on appointment → edit icon → enter case via drop-down within case bar

  ![6 Add Case Type Gi](/images/6_add_case_type.gif)
</Card>

**Create New Case Type**

<Card title=" ">
  **Create new case type:** The **only required field** is the Case Name. However, since insurance is a required field within the Appointments page, we recommend that insurance priority is also set.
</Card>
