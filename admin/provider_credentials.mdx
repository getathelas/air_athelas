Credentialing is a critical process that validates whether providers meet the requirements of insurance companies to deliver care under their plans. Traditionally, this information is fragmented, requiring manual lookups and repeated back-and-forth. Our credentialing matrix eliminates that friction by giving all stakeholders immediate visibility into credentialing statuses.

## Provider Credential set-up

### Adding a provider credential

<Card title=" ">
  1. Navigate to Provider Credentialing. This would be under the Utilities section
  2. Once you are on the page, click on Add Credentialing

  ![Pc 1 Pn](/images/pc_1.png)

  3. Search for Provider names. We also show the individual NPIs as well. Select one provider.
  4. Search for Insurance Companies. Select one of the insurance companies. You can search with insurance companies' name as well.
  5. Select Credentialing Type. You can select list of credentialing types per entry. Supported entries include
     - Unknown
     - Unknown
     - Group
     - Individual
     - Pending
     - None

  ![Pc2 Pn](/images/pc2.png)

  6. Based on the credentialing type selected, check the relevant values If provider is already credentialed,

  ![Pc3 Pn](/images/pc3.png)

  7. Click on Confirm to save your changes
  8. You can see the success message!
</Card>

### Bulk edit provider credentials

<Card title=" ">
  You can select multiple rows across different providers, insurance companies, and credentialing groups.

  1. You can select multiple rows on the table

  ![Pc4 Pn](/images/pc4.png)

  2. On the right side of the table, you can see the edit section. You have the ability to change the:

  5. You can see the success message!
</Card>

## Credentialing Group

Credentialing group allows you to bundle multiple insurance companies under one group, so when you get credentialed for a group, you can apply the same configuration.

<Danger>
  **Important Notes:**

  - An Insurance company can be part of multiple credentialing groups.
  - Groups are mutually exclusive with individual insurance selection in the Add Credential modal
  - You can select multiple groups when adding credentials to credential a provider with all insurances across those groups
</Danger>

### Creating Credentialing Groups

<Card title=" ">
  - Navigate to Provider Credentialing page (https://insights.athelas.com/provider-credentialing)
  - **Click the "+ Credential Group" split button** in the top right

    <img
      src="/images/pc9.png"
      alt="Pc9 Pn"
      title="Pc9 Pn"
      style={{ width:"49%" }}
    />
  - **Fill out the group details:**
    - Enter a group name (e.g., "Optum Group", "Medicaid Plans")
    - Add an optional description
    - Select insurance companies to include in the group
  - **Click "Save Group"** to create the group

    <img
      src="/images/pc10.png"
      alt="Pc10 Pn"
      title="Pc10 Pn"
      style={{ width:"60%" }}
    />
</Card>

### Editing Credentialing Groups

<Card title=" ">
  1. **Click the dropdown arrow** on the "Credential Group" button
  2. **Select "Edit Credential Group"**

     <img
       src="/images/pc11.png"
       alt="Pc11 Pn"
       title="Pc11 Pn"
       style={{ width:"47%" }}
     />
  3. **Choose the group** you want to edit from the dropdown
  4. **Modify the details** as needed (name, description, insurance companies)
  5. **Click "Save Group"** to update, or **"Delete Group"** to remove it
</Card>

### Adding Credentials Using Groups

<Card title=" ">
  1. **Click "Add Credentialing"** to open the credential modal
  2. **Select a provider**
  3. **Choose your credentialing approach:**
     - **Individual**: Select a specific insurance company, OR
     - **Group**: Select one or more credentialing groups
  4. **Set credential details** (type, status, cosigner requirements)
  5. **Click "Confirm"** - the system will automatically create individual credentials for each insurance in the selected groups
</Card>

### Deleting a credentialing group

<Card title=" ">
  You can also delete a credentialing group.

  **Steps:**

  1. Click on the credentialing group → down arrow icon → `Edit Credentialing group`
  2. Select the group to be edited.
  3. On the bottom of the drawer, you can see the `Delete credentialing group`.
  4. A confirmation modal appears. On click of `Delete Group`, the credentialing group would be Permanently deleted!
</Card>

### Using credentialing group as filters

<Card title=" ">
  1. Click the `+ Filter` button above the table. This will show various filters. Select `Credentialing Group`
  2. You can search for various groups.
  3. Click on each row to select/deselect the credentialing group.
  4. On click, the table would load the filtered rows.
</Card>

### Using credentialing groups to add credentials

<Card title=" ">
  1. Click on `+ Add Credentialing` button
  2. Select the provider name from the list of providers from the site.
  3. Select the credentialing group from the existing list of credentialing groups.
  4. Select the credentialing type. Based on the credentialing type, you would get the options to modify `Credentialed` and `Cosigner required`.
  5. Click on `Submit`.
  6. The provider credentialing entries for provider + insurance companies under the credentialing group would be created.
</Card>
