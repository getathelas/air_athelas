---
title: "Appointment Types"
---

You can create and manage custom appointment types, each with its own settings, clinical note structure, and optional CPT codes.

**Steps to Create a Custom Appointment Type:**

1. Navigate to the **Preferences** tab in the left navigation pane and open **Appointment Types**.
2. Click **\+ Appointment Type** in the top-right corner.
3. Enter the **Name**, **Duration**, and **Color** for the appointment type.
4. Select the **sections** to include (and their order).
   - You can choose to make them available for the Scribe to write into.
   - If you include **Measurements**, you can set default measurements to auto-populate.
5. (Optional) Assign **CPT codes** that will auto-populate for this appointment type.
6. Select a **default clinical note type**, then click **Create**.
7. You can edit or delete appointment types from the same tab at any time.
