This quality measure tracks the percentage of **visits** where an eligible clinician **attests to documenting, updating, or reviewing** the patient’s **current medications** using **all immediate resources available on the date of the encounter**.

## What the measure counts

For an eligible visit, your documentation is counted based on what you select in **Medication Review Attestation**:

### <Icon icon="circle-info" iconType="duotone" color="#F9345F" size={23} />  Performance categories

- **Reviewed Current Medications**: **Performance met**
- **Did Not Review Current Medications**:
  - **Emergency/Critical Situation**: **Exception**
  - **Other Reason**: **Performance not met**

**Note:** To support accurate reporting, you should complete the attestation for **every eligible encounter** in the measurement period (each visit is its own episode).

## How to document Current Medications in the Visit Note

### <Icon icon="clipboard-check" iconType="duotone" color="#F9345F" size={23} />  Complete the Medication Review Attestation

**To document this measure:**

1. Open the patient’s **Visit Note**.
2. In the left sidebar, go to **Medications**.
3. Add the **Medications** and **Medication Review Attestation** sections (if they are not already on the note).
4. Under **Medication Review Attestation**, select the appropriate **Medication Review Status**:
   - Select **Reviewed Current Medications** if you reviewed the patient’s current medication list.
   - Select **Did Not Review Current Medications** if you did not review the list, then select a reason.

**Verify it saved:** Before you submit the note, confirm the **Medication Review Attestation** section shows the status you selected.

### FAQ

<Accordion title="Where do I document the measure in Air?">
  In the patient’s **Visit Note**, go to **Medications** and add the **Medication Review Attestation** section. Your selection in **Medication Review Status** is what is reported.
</Accordion>

<Accordion title="What should I select if the patient has no medications?">
  If you reviewed the patient’s medication list and confirmed they have no current medications, select **Reviewed Current Medications**. This indicates you performed the review.
</Accordion>

<Accordion title="When should I use Emergency/Critical Situation?">
  Select **Emergency/Critical Situation** only when the visit circumstances prevent completing a medication review and documenting it would not be clinically appropriate at that time. This is counted as an **exception** for the measure.
</Accordion>

<Accordion title="Do I need to complete this for every visit?">
  Yes—complete the **Medication Review Attestation** for every **eligible encounter** during the measurement period. Each visit is counted as its own episode.
</Accordion>
