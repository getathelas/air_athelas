Documenting a patient’s **current medications** helps your practice stay clinically accurate and supports reporting requirements that score medication review on a per-visit basis.

## About the “Current Medications” measure

### <Icon icon="clipboard-check" iconType="duotone" color="#F9345F" size={23} />  What the measure is counting

This measure looks at the percentage of **visits** where the eligible clinician **attests** that they documented, updated, or reviewed the patient’s list of **current medications**, using all immediate resources available on the date of the encounter.

**Note:** Each eligible encounter is scored independently (each visit is its own “episode”).

## Document Current Medications in the Visit Note

### <Icon icon="notes-medical" iconType="duotone" color="#F9345F" size={23} />  Add the required sections (one-time per note, if missing)

**To add the sections:**

1. Open the patient’s **Visit Note**.
2. In the left sidebar, go to the **Medications** category.
3. Add these sections (if they aren’t already on your note):
   - **Medications**
   - **Medication Review Attestation**

<Tip>
  ✨**Smart Tip:** If you don’t see these sections as options, your chart note template may need to be updated to include them.
</Tip>

### <Icon icon="comment-check" iconType="duotone" color="#F9345F" size={23} />  Select the Medication Review Status

1. In **Medication Review Attestation**, select the appropriate **Medication Review Status**.
2. Choose the option that matches what happened for the visit:
   - Select **Reviewed Current Medications** if you reviewed the patient’s current medication list. This is scored as **Performance Met**.
   - Select **Did Not Review Current Medications** if you did not review current medications. This is scored based on the reason you select:
     - **Emergency/Critical Situation** → scored as an **Exception**
     - **Other Reason** → scored as **Performance Not Met**

## How scoring works (quick reference)

| **What you select in Medication Review Attestation** | **How the visit is scored** |
| :--- | :--- |
| **Reviewed Current Medications** | **Performance Met** |
| **Did Not Review Current Medications** + **Emergency/Critical Situation** | **Exception** |
| **Did Not Review Current Medications** + **Other Reason** | **Performance Not Met** |

### FAQ

<Accordion title="Where do I document Current Medications for this measure?">
  In the **Visit Note**, go to the **Medications** category and complete **Medication Review Attestation** (and the **Medications** section as needed).
</Accordion>

<Accordion title="I reviewed the medication list but didn’t change anything. What should I select?">
  Select **Reviewed Current Medications** as long as you reviewed the patient’s current medication list using the information available at the time of the visit.
</Accordion>

<Accordion title="When should I select Emergency/Critical Situation?">
  Select **Emergency/Critical Situation** only when an urgent clinical situation prevented you from reviewing current medications during the encounter. This is scored as an **Exception**.
</Accordion>

<Accordion title="Does this measure apply to every visit for a patient?">
  It applies to **eligible encounters** during the measurement period, and each eligible encounter is scored independently (one visit = one episode).
</Accordion>
