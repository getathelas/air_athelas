## Set your Calendar Settings

<Accordion title="Appointment Type preferences">
  Appointment Types reference unique type of visit your practice conducts.

  Our Appointment Type configuration allows you to construct an appointment that represents the exact conditions you would like to see. For instance, you and your practice have the ability to configure the Duration, the specific Color, the Dynamic EHR Template and which Components you would like to see.

  Clicking on the top right icon “\+ Add Appointment Type” creates a completely new appointment type. On the other hand, the Pencil icon on the left hand side of the screen allows you to edit any pre-existing appointment types.

  ![Appt Types Web](/calendar/appt_types.webp)

  When creating a new appointment type, the following information will be available for configuration:

  - Appointment Type Name
  - Appointment Duration
  - Appointment Color
  - Section selection and order
  - Option to sync duration with time in clinic - This means that the time spent within the actual appointment with the patient will be reflected within the calendar.

  <img
    src="/calendar/select_appt_sec.webp"
    alt="Select Appt Sec Web"
    title="Select Appt Sec Web"
    style={{ width:"58%" }}
  />

  Sections are divided between default and custom sections. Default sections are hard coded components that are created by Athelas. Custom sections are the templates that the users created. Any number of sections can be selected and ordered in any way. This order is what will appear on the chart note.

  <img
    src="/calendar/order_appt_type.webp"
    alt="Order Appt Type Web"
    title="Order Appt Type Web"
    style={{ width:"67%" }}
  />

  If measurements are a selected section indicated within the appointment type, you will be able to specify the specific measurements you’d like to see each time this appointment type is scheduled for a patient.

  <img
    src="/calendar/appt_type_meas.webp"
    alt="Appt Type Meas Web"
    title="Appt Type Meas Web"
    style={{ width:"59%" }}

  For **Clinical Note Types** beyond the Initial Evaluation you are able to select the specific Sections you would like to carry over, hence, if Measurements are not necessary beyond the Initial Evaluation you can configure accordingly
</Accordion>

<Accordion title="Calendar Preferences">
  **Set calendar timings:** You can configure the calendar start and end times. Head to Preferences → Calendar → Start & End Times. Here you can select the starting time and an the ending time.

  **Default timings:** By default, the calendar start and end times are set to display 06:00am to 08:00pm and this is applied at a site level (i.e. all facilities adopt the same times). You can choose to set one site-wide start & end time or set specific times for specific facilities.

  ![Cal Pref Web](/calendar/cal_pref.webp)

  **Unique timings across specific facilities:** If you wish to specify unique start & end times to specific facilities, select the “Facility” option and a table should appear below.

  On this table, selecting any facility will make its start & end time controls appear on the right. Select multiple facilities to modify multiple facilities at once. Once complete, you updates should already be live in the calendar.

  ![Cal Times Web](/calendar/cal_times.webp)

  \*\*Calendar Refresh Mode: \*\*You now have the ability to select how how quickly the calendar updates. Standard refresh mode means that the calendar will only update if the browser is refreshed.

  However, real-time refresh mode will be updated in real-time. E.g. if multiple front-desk staff are altering the calendar (ex: creating, cancelling, editing, checking in appointments) the calendar will immediately reflect those changes.
</Accordion>
