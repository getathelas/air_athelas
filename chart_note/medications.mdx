### Add New Medication

<Card title=" ">
  From Patient's Profile \> Medications you can:

  - View Historic Medications
  - View Prescribed Medications
  - Add / Edit Prescribed Medications

  You can search for the drug name and filter the medications by date, activity, type, pharmacy status _(sent, pending, received, unknown, failed)_

  ![Medications Page Web](/images/medications_page.webp)

  **To Add a new Medication:**

  - Click **Click New:** Select Pharmacy Order, Internal Record or Historic Record

    <img
      src="/images/create_rx.webp"
      alt="Create Rx Web"
      title="Create Rx Web"
      style={{ width:"40%" }}
    />
  - Select the relevant appointment

  ![46 Rx Open Gi](/images/46_rx_open.gif)

  - Select the\*\*+ icon \*\* to the right to select a Pharmacy (if a favorite pharmacy is selected it will auto-populate)
  - Search for a pharmacy by typing the name, pin code, distance radius, pharmacy type (Retail, Mail Order)
  - Star a pharmacy to **mark it as favorite**  or preferred

  ![47 Rx Pharmacy Choice Gi](/images/47_rx_pharmacy_choice.gif)

  - Select a medication

  <Danger>
    **Smart Tip:** If a Generic alternate is available - you can choose to update the medication selected.
  </Danger>
  - Check the box: I understand these alerts and will proceed
  - Enter the prescription SIG or select from the saved SIGs

  <Danger>
    **Smart** \*\*Tip: \*\*Click Check with AI to get a well worded SIG statement. You can choose to accept or delete the suggestion.
  </Danger>
  - Add the duration, dispense quantity, refills
  - Select \*\*Dispense as Written \*\*if you would like the pharmacy to dispense the medication as written in the prescription order
  - Add internal notes, pharmacy notes, diagnosis codes from the appointment, earliest fill date
  - Click \*\*Proceed to Review \*\*to save the medication

  ![48 Rx Prescribe Gi](/images/48_rx_prescribe.gif)

  \*\*Note: \*\*You can also add medication into the Chart Note as an additional section.
</Card>

### View Medication History

<Card title=" ">
  You can view medication history two ways:

  1. Manually add historic medications from \*\*Create New \*\*and view in the Historic Medications Tab
  2. View Medication History on SureScripts by selecting \*\*Medication History \*\*on the top right. Select time range, acknowledge patient's consent and click Submit

  ![43 Med History Gi](/images/43_med_history.gif)

  ![Rx History Web](/images/rx_history.webp)
</Card>

### Create Custom SIGs

<Card title=" ">
  View Saved SIGs on Preferences Tab \> Medications. You can search by Drug Name or Active / Inactive status

  ![45 New Sig Gi](/images/45_new_sig.gif)

  Select the pencil icon to update a SIG and select the toggle icon to mark a SIG as inactive.

  To create a new SIG:

  - Click **\+add SIG**
  - Select the drug name
  - Add a label, SIG text and mark state as Active
  - Click Save SIG

  <img
    src="/images/create_sig.webp"
    alt="Create Sig Web"
    title="Create Sig Web"
    style={{ width:"40%" }}
  />
</Card>

### Manage Pharmacy Requests

<Card title=" ">
  Navigate to Pharmacy Requests on the left navigation bar. From here you can do two things:

  ![Pharmacy Req1 Web](/images/pharmacy_req1.webp)

  **Respond to Renew / Change Requests [Requests Tab]**

  - Click Review on the right of a request
  - Review Rx Change Request Details. You can make modifications, Approve, Replace or Deny the request

  ![Request Pharma Approve Web](/images/request_pharma_approve.webp)

  **Link Unrecognized Patients [Unrecognized Patients Tab]**

  - Click Review on the right of a request
  - Select an existing patient from the drop down list
  - Click Map to Patient at the bottom
  - The request will move from Unrecognized Patients to Requests

  ![Linkpt Pharma Request Web](/images/linkpt_pharma_request.webp)
</Card>
